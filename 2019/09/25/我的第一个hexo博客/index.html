1. 我的第一篇用Typora编写的博客

1.1. 第一步在MyBlog文件夹下生成一个新的.md文件

在MyBlog 文件夹下右击，在弹出对话框中选中Git Bash here ,将会弹出git控制台，在控制台中输入

hexo n "我的第一个hexo博客"。等待一会，会在MyBlog文件下的source文件夹下的_posts下生成我的第一个博hexo博客.md文件。

1.2. 第二步，直接打开这个文件，使用Typora编辑

1.3. 第三步，保存编辑好的文件，推送到本地

退回到MyBlog目录下，右击鼠标，点选Git Bash Here，在命令行窗口输入

hexo clean hexo g hexo s

这样就可以在localhost:4000 查看了

1.4. 第四步，推送到github.io上

先使用 ctrl+c打断，之后用命令 hexo d
